# Skill: การ Query ฐานข้อมูล (MSSQL ผ่าน MCP)

## ขั้นตอนบังคับ
1. เรียก `get_database_context` **ก่อนเสมอ** เพื่อดู schema, ความสัมพันธ์ และคู่มือ T-SQL
2. เขียน T-SQL ให้ถูกหลัก SQL Server:
   - ใช้ `TOP n` ไม่ใช่ `LIMIT n`
   - ใช้ `GETDATE()` ไม่ใช่ `NOW()`
   - ใช้ `[]` ครอบชื่อคอลัมน์/ตารางที่มีอักขระพิเศษ
3. ส่ง query ให้ `execute_query_tool`
4. ถ้าต้องการแค่ดูตัวอย่างข้อมูล ใช้ `preview_table` แทนการเขียน query เอง

## หลักการตอบ
- สรุปเป็นภาษาไทย พร้อมตารางและข้อสังเกตเชิงธุรกิจสั้น ๆ
- ถ้า query ผิด/ไม่มีข้อมูล ให้ตรวจ schema ใหม่จาก context แล้วลองอีกครั้ง อย่าเดาชื่อคอลัมน์
