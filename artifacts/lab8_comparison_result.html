### 🔍 วิธีตรวจสอบความถูกต้อง (Verification Process)

การตรวจสอบใช้ **2 Query เปรียบเทียบกัน** ดังนี้:

| ขั้นตอน | Query ที่ใช้ | วัตถุประสงค์ |
|---|---|---|
| **Query A** | `SELECT TOP 1 ... ORDER BY COUNT DESC` | หาแผนกอันดับ 1 โดยตรง |
| **Query B** | `SELECT ... GROUP BY department ORDER BY COUNT DESC` (ไม่มี TOP) | แสดงทุกแผนกพร้อมกัน |

**หลักการตรวจสอบ:**
1. ✅ **ค่า COUNT ตรงกัน** — Query A ได้ `เทคโนโลยีสารสนเทศ = 5` ตรงกับแถวแรกของ Query B
2. ✅ **ห่างจากอันดับ 2 อย่างชัดเจน** — อันดับ 2 (การตลาด/ทรัพยากรบุคคล) มีเพียง 4 คน ต่างกัน 1 คน ไม่มีการ tie ที่อันดับ 1
3. ✅ **จำนวนรายชื่อสอดคล้อง** — Query ดึงรายชื่อพนักงานได้ **5 แถว** ตรงกับ COUNT = 5 พอดี
4. ✅ **เงื่อนไข `status = N'ปฏิบัติงาน'`*
